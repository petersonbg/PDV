# Frontends

## PDV Desktop (Electron + React)
- UI otimizada para teclado/leitor e fluxo de venda rápido.
- Executa em Electron com React + Vite.
- Script `dev:electron` inicia Vite e a janela Electron apontando para o dev server.

## Admin Web (React)
- SPA para cadastros, estoque, financeiro e relatórios.
- Pode ser hospedada como app estático consumindo a API FastAPI.

Estruturas de exemplo incluem componentes mockados para demonstrar layout e comunicação futura com a API.
